Welcome to Roy's .NET Dating

A full-stack dating application showcasing modern web development with .NET and Angular

About This Project

This is a comprehensive portfolio project demonstrating full-stack development capabilities using the latest technologies in the .NET and Angular ecosystems. Built as a modern dating application, it showcases real-world features and industry best practices.

The application combines a robust .NET 8 backend with an Angular 19 frontend, featuring real-time communication, secure authentication, and a responsive user interface that works seamlessly across all devices.

Technology Stack

Backend

  • .NET 8: Latest long-term support framework
  • Entity Framework Core: Object-relational mapping
  • PostgreSQL: Production-grade database
  • SignalR: Real-time bidirectional communication
  • JWT Authentication: Secure token-based auth
  • AutoMapper: Object-to-object mapping

Frontend

  • Angular 19: Latest standalone components
  • TypeScript: Type-safe development
  • Bootstrap 5: Responsive UI framework
  • RxJS: Reactive programming
  • Signal-based state: Modern state management
  • Progressive design: Mobile-first approach

Key Features

User Authentication

Secure registration and login with JWT tokens, password hashing, and role-based authorization for admins and moderators.

Photo Management

Upload and manage profile photos with Cloudinary integration, set main photos, and moderate user-submitted content.

Real-Time Messaging

Instant messaging using SignalR with message threads, read receipts, and online presence indicators.

Likes & Matching

Like other users, view who liked you, and discover mutual matches with efficient database queries and pagination.

Advanced Filtering

Filter members by age, gender, location, and other criteria with optimized queries and client-side caching.

Admin Panel

Comprehensive admin interface for user management, photo moderation, and role assignment with proper authorization.
